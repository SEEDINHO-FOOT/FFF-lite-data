# FFF-lite-data

Compilation **légère** de données football au format JSON : matchs, scores, classements
et index de recherche — **structuration, organisation et indexation** réalisées par
Koffi Frederic SESSIE à partir de données publiques issues du football français
(fédération, ligues et districts).

> Version *light* : couverture complète, rapide à mettre à jour. Les détails par match
> (buteurs, cartons, arbitres, stade) ne sont pas inclus dans cette version.
